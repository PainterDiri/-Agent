# 产品推荐规则

## 一、推荐目标

Agent 不是为了把所有商品都介绍一遍，而是帮游客降低选择成本。每次对话最终只能给出一个结果：

```text
一个好运主题 × 一种商品形式 = 一个明确商品
```

例如：

- 鲁智深·拔掉烦恼成品香囊。
- 李逵·快乐加倍 DIY 香囊。
- 水浒随机福袋。
- 张顺·顺风顺水水浒特色头巾。

## 二、好运人物点定规则

| 需求信号 | 推荐人物 | 解释 |
|---|---|---|
| 压力、烦恼、焦虑、想放松 | 鲁智深 | 用“拔掉烦恼”表达放下和轻装上阵 |
| 孩子、亲子、朋友、想开心 | 李逵 | 用“快乐加倍”表达同行欢乐 |
| 旅行、工作、学业、办事、出行 | 张顺 | 用“顺风顺水”表达行动顺遂 |
| 父母、长辈、家庭、平安、送礼 | 安道全 | 用“安到全家”表达家庭祝愿 |

四位人物不会作为首页选择器出现。Agent 根据游客在对话中说出的愿望点定一位人物，作为最终好运主题；若游客主动提到喜欢某位人物，则优先尊重这一偏好。

## 三、商品形式规则

### 成品香囊

出现以下信号时推荐：

- 赶时间。
- 直接带走。
- 不想动手。
- 送父母、长辈或朋友。
- 希望结果明确，不喜欢随机。

### DIY 香囊

出现以下信号时推荐：

- 带孩子或亲子同行。
- 喜欢动手、亲手制作。
- 愿意停留约五分钟。
- 重视共同完成的纪念意义。

### 随机福袋

出现以下信号时推荐：

- 喜欢惊喜、盲盒或拆袋。
- 对兵器、贴纸、驱蚊贴、书签感兴趣。
- 儿童想要多种小物。
- 不介意袋内具体内容随机。

### 水浒特色头巾

出现以下信号时推荐：

- 喜欢穿戴、拍照或角色装扮。
- 正在户外游览，希望商品可以立即使用。
- 想要醒目、可反复使用的水浒纪念品。
- 主动提到头巾、帽饰或水浒人物印花。

## 四、冲突时的优先级

1. 游客明确说出的商品偏好。
2. 是否带儿童和是否愿意等待。
3. 送礼还是自用。
4. 对话内小术形成的好运联想。
5. Agent 根据模糊关键词推断的人物。

小术只负责增加故事感，不应推翻游客真实需求。例如颜色或方位联想到李逵，但游客明确要送父母且希望全家平安，应推荐安道全成品香囊。

## 五、推荐表达

推荐需要包含：

1. 商品全名。
2. 一句与游客回答有关的理由。
3. 一句人物祝福。
4. 提示将结果给摊位工作人员看。

禁止包含：

- “这两个都适合你”。
- “再加一点钱可以买更好的”。
- “今天不买就没有了”。
- “买了就一定能转运”。
- “这个能治病、助眠、保考试通过”。

## 六、对话轮数

理想流程为 2—4 轮：

1. 为谁问。
2. 最想获得哪类祝福。
3. 喜欢直接带走、DIY 还是随机惊喜。
4. 必要时补问时间或送礼对象。

游客已经在一句话中给齐信息时，应直接给结论，不重复提问。
